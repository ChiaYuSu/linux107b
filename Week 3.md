## 正則表達式常見特殊字符
| 字符 | 含意 |
| ---- | ---- |
| `^` | 指向一行的開頭 |
| `$` | 指向一行的結尾 |
| `.` | 任意單個字符 |
| `[]` | 字符範圍 |
> 如果想將上述字符做為普通字符，就需要在他們前面加上 `\` 字符。舉例來說，如果又使用 `$` 字符，需要在 `$` 前面加上跳脫字元符號 (`\`) ，寫成 `\$`。
